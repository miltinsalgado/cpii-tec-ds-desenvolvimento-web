Bem-vindo!

Olá! Seja bem-vindo ao Zen Garden ICC! O objetivo deste desafio é apresentar as diversas possibilidades que a utilização de CSS nos oferece, ao propor que cada pessoa altere completamente a aparência desta página utilizando apenas CSS.

Como participar?

Baixe o arquivo base HTML e o CSS, altere o CSS à vontade e, ao terminar, envie para o formulário APENAS o arquivo zipado contendo o arquivo .css e qualquer imagem ou fonte web que você tenha utilizado/citado no CSS. Posteriormente, eu postarei cada uma das páginas no site. Seja criativo!
